Getting started

To get started download the template from the config page and upload it to your google drive. Ensure that it gets converted to a google Docs format when uploading. After that the spreadsheet should show up in the confg page. Click the checkbox of one or several spreadsheets and save your settings. The app will only display selected spreadsheets.

The WebFlipFlapp File Format

The format should be self explaining. Please leave the first two rows untouched. The spreadsheet has to comply with the following things:

  • The first column, first row cell has to be WFF:FLYSTOCK
  • New boxes can be created in a spreadsheet by copying the two box header columns. Boxes need to be seperated by at least two empty rows
  • Do not change the format of the two box header columns. (you can/should change the boxname)
  • Box names have to be unique in a spreadsheet.

There are some additional things that you can use:

  • put WFF:IGNORE somewhere in the first column of your spreadsheet, and the app will ignore everything after that row.
  • add WFF:HOSPITAL, WFF:WARNING or WFF:DEAD to the Mod column of a fly to hightlight it in the flies view

Getting labels

Labels can be generated in several different ways. You can download labels in the boxes page for single boxes or whole flystocks. Another option is to download labels in the flies view. You can either create them for all displayed flies (the ones filtered by your search terms) or select flies and only create them for the selected ones. Dont forget to choose the correct label template in the config menu!

  • Hint: searching while selecting flies is possible! The app remembers your selection
  • Hint: you can save the environment and use old labelsheets by specifying how many empty labels should be on the pdf with the skip option
